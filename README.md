WeatherIQ

A simple web app that displays the current weather for 6 fixed cities.

Features :

- Shows weather details (temperature, condition, humidity, wind speed) for 6 present cities
- Clean and minimal design using HTML, CSS, and JavaScript
- Uses a weather API to fetch live data for each city

Technologies Used :

1.HTML
2.CSS
3.JavaScript
4.Weather API (e.g., OpenWeatherMap)

How to Use :

- Open the `index.html` file in any modern web browser.
- The app automatically displays weather information for the 6 cities.

Setup:

- No installation required.
- To update API key, replace it in the JavaScript file if needed.

Future Improvements :

- Add search functionality for any city
- Show forecasts and more weather details
- Improve UI and add animations
